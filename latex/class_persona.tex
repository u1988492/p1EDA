\doxysubsection{Detailed Description}
Clase que representa a una persona con atributos como año de nacimiento, estudios y nacionalidad. 

La clase {\ttfamily \doxylink{class_persona}{Persona}} se utiliza para almacenar y gestionar la información relevante de un habitante, incluyendo su nivel de estudios y nacionalidad.
